# nyss-laba1
## Шифр Цезаря
WPF-приложение, позволяющее осуществить шифровку введенного/загруженного пользователем текста с последующей возможности сохранения шифрованного текста.

Входящий/исходящий формат файла - .txt <br>
Шифровке подлежат символы русского алфавита и цифры.

![alt text](https://s4.gifyu.com/images/ezgif-3-39518222ad59.gif)

Проект содержит несколько Unit-тестов на проверку правильности шифровки/дешифровки.

<b>Возможности:</b>
1) Возможность шифровки/дешифровки текста
2) Выбор шага шифровки/дешифровки текста
3) Выбор направления шифровки/дешифровки
4) Несколько видов алфавитом для шифровки, а именно: <br>
    -общий алфавит для русских букв и цифр (Всего 43 символа, сначала буквы, затем цифры (А...Я,1...0))<br>
    -раздельный алфавит для русских букв и цифр (33 символа + 10 символов ((А...Я),(1...0)))
5) Возможность загрузки исходного и сохранения выходного файлов
6) Возможность непосредственного ввода текста в интерфейсе программы
